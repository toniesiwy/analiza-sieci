KROKI URUCHOMIENIA (MacOS):

1. python3 -m venv scapy_env      #utworzenie wirtualnego srodowiska

2. source scapy_env/bin/activate     #uruchomienie wirtualnego srodowiska

3. pip install scapy     #instalacja niezbednego pakietu do skanowania 

4. python analiza.py     #wywolanie analizy w folderze w którym znajduje się skrypt


*Jezeli system umozliwia Ci bezposrednia instalacje biblioteki 'scapy', wykonaj tylko ostatnie 2 polecenia w celu wywołania analizy sieciowej
